Proyecto

Nombre del proyecto: Queque Girasol

Resumen del proyecto

Cliente

Queque Girasol es una pastelería dedicada a la elaboración de queques y postres personalizados para diferentes celebraciones y ocasiones especiales.

Objetivo

Diseñar un prototipo de alta fidelidad para una página web que permita a los usuarios conocer los productos de la pastelería y personalizar su propio pastel de forma intuitiva, atractiva y sencilla.

Público objetivo

El sitio está dirigido a personas que desean adquirir pasteles personalizados para cumpleaños, aniversarios, eventos familiares y otras celebraciones. Se busca ofrecer una experiencia amigable para usuarios de todas las edades.

Solución propuesta

Se desarrolló un prototipo interactivo utilizando Figma, acompañado de un sistema de diseño consistente, investigación de usuarios, documentación del proceso, evaluación de accesibilidad y una guía de estilos. El sistema de personalización permite seleccionar diferentes opciones para construir un pastel según las preferencias del usuario.

Estructura del proyecto

01_Investigación

Contiene toda la documentación relacionada con la investigación previa al diseño.

* Análisis del público meta.
* Entrevista al cliente y resultados.
* Racional creativo.
Esto se encuentra en un solo archivo

02_Sketches_Wireframes_y_Mockups

Incluye el proceso de diseño desde los primeros bocetos hasta los mockups.

* Sketches.
* Wireframes.
* Mockups.
* Evidencias del proceso.

03_Prototipo_Figma

Contiene el prototipo final desarrollado en Figma.

* Archivo .fig del proyecto.
* Enlace compartido al archivo de Figma.
* Especificaciones técnicas.
* Archivo con la URL del repositorio.

04_Guía_de_Estilo_y_Diseño

Incluye toda la documentación del sistema de diseño.

* Design System.
* Paleta de colores.
* Tipografías y Iconografía.

05_Accesibilidad

Contiene la evaluación realizada sobre accesibilidad.

* Reporte WCAG 2.1 AA.
* Mejoras implementadas.

06_Presentación

Incluye los recursos utilizados durante la exposición del proyecto.

* Presentación en PDF o PowerPoint.
* Guion de presentación.

---

Instrucciones de navegación

1. Inicie revisando la carpeta *01_Investigación* para comprender el análisis realizado antes del diseño.

2. Continúe con *02_Sketches_Wireframes_y_Mockups* para observar la evolución del proyecto.

3. Abra la carpeta *03_Prototipo_Penpot_o_Figma* para visualizar el prototipo interactivo mediante el archivo .fig o el enlace compartido.

4. Consulte la carpeta *04_Guía_de_Estilo_y_Diseño* para conocer los archivos que fueron implementados en el proyecto.

5. Revise *05_Accesibilidad* para verificar las evaluaciones y mejoras aplicadas.

6. Finalmente, consulte la carpeta *06_Presentación* para acceder a la presentación y demás materiales utilizados durante la exposición.
